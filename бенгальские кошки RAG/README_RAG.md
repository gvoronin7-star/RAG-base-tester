## Обзор

Эта RAG-база данных содержит векторизованные документы, подготовленные для использования в системах семантического поиска и генерации ответов на основе контекста.

**Параметры базы данных:**

- **Название:** бенгальские кошки RAG
- **Модель эмбеддингов:** text-embedding-3-small
- **Размерность:** 1536
- **Количество векторов:** 210
- **Количество чанков:** 210
- **Тип БД:** FAISS IndexFlatIP

```

    ├── 2025-12-04 Бенгал информация Perplexity.pdf
```

### Описание файлов

**index.faiss**

Индекс FAISS, содержащий векторные представления всех чанков документов. Используется для быстрого поиска ближайших векторов.

**dataset.json**

JSON-файл с полным набором чанков документов. Каждый чанк содержит:

```json
{
  "text": "Текст чанка...",
  "type": "paragraph",
  "context": {},
  "metadata": {
    "source": "document.pdf",
    "page_number": 1,
    "chunk_id": "chunk_000",
    "timestamp": "2026-02-18T12:00:00",
    "chunk_type": "paragraph",
    "char_count": 480,
    "word_count": 80
  }
}
```

**metadata.json**

Метаданные индекса, включая параметры модели и статистику.

**source_documents/**

Папка с исходными PDF-файлами, использованными для создания базы данных.

## Содержание базы данных

База данных содержит информацию о бенгальских кошках:

- **История породы** — происхождение, развитие, селекция
- **Характер и поведение** — темперамент, особенности психологии
- **Окрасы и стандарты** — признанные окрасы, требования стандарта
- **Уход и содержание** — питание, груминг, условия содержания
- **Здоровье** — наследственные заболевания, профилактика
- **Особенности породы** — отличия от других пород, уникальные черты
