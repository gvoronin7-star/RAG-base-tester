## Нумерация версий

Проект следует [Semantic Versioning](https://semver.org/spec/v2.0.0.html):

- **MAJOR** (2.x.x) — несовместимые изменения API
- **MINOR** (x.5.x) — новая функциональность, обратно совместимая
- **PATCH** (x.x.0) — исправление ошибок, обратно совместимое
